My contributions:

  • Homepage - I designed and develped the homepage of the website where the splash screen leads to. The navbar was also a part of it and I implemented a cursor parallax effect to make the website more interactive. Homepage includes links to the 4 individual pages of related articles and some other shortcuts in the website too.
  • Navbar - As a part of the home page, I designed the initial navbar of the website and let others implement it. It includes a hover effect with a line animation and an active class to show what page is active too. Also it contains a theme switcher drop down menu to switch the color theme of the entire website instantly that will be saved even after the session ends.
  • Cursor - I designed the cursor of the website and implemented it with a parallax effect to make the website more interactive. It is a custom cursor that is not visible on the splash screen. It is only visible on the other pages of the website.
  • Comments - The comments forum which takes user feedback was also implemented by me and it sends that data to our team google group's email address throut the formsubmit.co service to get the feedback to all the members of the team and so the user does not need to provide personal emails to us.
  • What is IMAX - I wrote the article about IMAX and its history and also designed the page as the related article page.
  • style.css - I designed the style.css file of the website and also the theme-cyan.css file which is the default theme of the website which will be loaded by a java script code in the index.html file from local storage of the browser is saved.
